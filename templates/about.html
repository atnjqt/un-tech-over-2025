About Page

This dashboard provides a visual representation of geospatial data across various countries referenced in the OSM model. You can filter the data by country and location types to customize your view.

Building out the Giga-Spatial platform to improve integration and accessiblity for OSM Handler along with reviewing the HDX Handler, which the team identified could be improved by setting up a dedicated UNHCR handler to include displaced and refugee populations. To get Ahead of the Storm, the team aims to demontrate value for UN members and policy makers in identifying risk for children that are displaced or residential along with Open Street Maps and Weather Data

Leveraging Python Flask for a simple web application to visualize and filter on data across OSM & UNHCR. The team has formed the main Giga-Spatial repository accessible here: Giga-Spatial and efforts were made to port this to Linux in a Docker containerization
